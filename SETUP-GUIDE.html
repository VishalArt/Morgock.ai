STEP 1

🌐 Open Google Apps Script

  1. Go to script.google.com
  2. Click + New project
  3. Click on the title "Untitled project" → rename it to morgock.ai Forms
  4. Delete everything inside the code editor (Ctrl+A → Delete)
STEP 2

📋 Paste the Script

  1. Open the file google-apps-script.js from your website folder in VS Code
  2. Select all (Ctrl+A) and copy (Ctrl+C)
  3. Go back to Apps Script editor → paste (Ctrl+V)
  4. Press Ctrl+S to save

💡 The script already has your 3 Sheet IDs hardcoded — you don't need to change anything in it.

STEP 3

🚀 Deploy as Web App

  1. In Apps Script, click Deploy (top right) → New deployment
  2. Click the ⚙️ gear icon → select Web app
  3. Fill in:
    • Description: morgock.ai Form Handler v1
    • Execute as: Me (your Google account)
    • Who has access: Anyone ← very important!
  4. Click Deploy
  5. A popup says "Authorization required" → click Authorize access
  6. Choose your Google account → click Allow
  7. You'll see a Web app URL — looks like:
https://script.google.com/macros/s/AKfycbXXXXXXXXXXXXXXXXXXXX/exec

⚠️ Copy this URL right now — you need it in Step 4!

STEP 4

🔗 Add URL to your website

  1. Open js/forms.js in VS Code
  2. Find line 6 — it says:
var SCRIPT_URL = "YOUR_SCRIPT_URL_HERE";

Replace with your actual URL:

var SCRIPT_URL = "https://script.google.com/macros/s/AKfycbXXXXX/exec";
  1. Save the file (Ctrl+S)
  2. Re-deploy to Vercel (drag the folder again, or push to GitHub)

✅ Done! All 4 forms now send data directly to your Google Sheets in real time.

STEP 5 — VERIFY

✅ Test it works

  1. Go to your live website → open the Demo page
  2. Fill in a test name + your email → click Request My Demo
  3. You should see a green toast: 🎉 Demo booked!
  4. Open your Demo Requests sheet — your row should appear!

💡 Set up email alerts: In any sheet → Tools → Notification Rules → Any changes → Email me immediately. You'll get a Gmail alert the moment someone fills a form.

TROUBLESHOOTING

🔧 If something doesn't work

  • Data not appearing in sheet?
    Make sure "Who has access" is Anyone — not "Anyone with Google account"
  • Toast says "Setup needed"?
    You haven't pasted the Script URL into js/forms.js yet — go back to Step 4
  • Changed the script and it stopped working?
    Every time you edit the script, go to Deploy → Manage Deployments → create a New Version
  • Permission error during authorization?
    Click "Advanced" → "Go to morgock.ai Forms (unsafe)" → Allow. This is normal for new scripts.
  • Want to test the API directly?
    Paste your Script URL into a browser. You should see: morgock.ai Form API is live
